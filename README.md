\## 📋 Project Overview



Homeland Jobs is a fully responsive job listings platform built as part of the HEH Developer Technical Assessment. This frontend application allows job seekers to browse, search, filter, and sort job opportunities, as well as submit proposals for specific jobs through an interactive modal form.



\### Key Features at a Glance



\- 🔍 \*\*Real-time Search\*\* with instant filtering

\- 🎯 \*\*3 Working Filters\*\* (Category, Location, Budget Range)

\- 📊 \*\*Smart Sorting\*\* (Newest, Budget High-Low, Budget Low-High)

\- 📱 \*\*Fully Responsive\*\* (3-col desktop, 2-col tablet, 1-col mobile)

\- 💼 \*\*12 Job Cards\*\* with complete job information

\- 📝 \*\*Job Detail Modal\*\* with proposal submission form

\- ✅ \*\*Inline Form Validation\*\* (no browser alerts)

\- ⏳ \*\*Loading Skeleton\*\* (1.5s simulated delay)

\- 🔄 \*\*Error State\*\* with Retry button

\- 🔒 \*\*Security Features\*\* (blocks inspection, right-click, DevTools)
